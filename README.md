## Funcionalities

- Create item
- Edit item
- Delete item 
- Check item


## Clone this project

```bash
  git clone git@github.com:flubyGit/angular-lifecycle.git
```

## Enter project
```bash
  cd angular-lifecycle
```

## Install dependencies

```bash
  npm install
```

## Run server

```bash
  npm run start
```
